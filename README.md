# Chogan Catalogue

Private reference catalogue for browsing Chogan perfume references and body care products.

The 168-row `catalogue_data.csv` is the catalogue source of truth. The browser imports
its generated equivalent from `catalogue_data.js`; both files must remain row-for-row
and field-for-field identical. Each record keeps legacy and current Chogan codes
separately, preserves documented references, and labels unverified or note-based
suggestions with an explicit verification status and confidence.

The pre-merge site is preserved as `index.backup-2026-06-28.html`.

Open `index.html` locally or publish the repository with GitHub Pages.
